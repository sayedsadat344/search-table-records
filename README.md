## About THIS APP

THIS IS A SIMPLE RECORD SEARCH APP, DEVELOPED IN LARAVEL.

#HOW TO RUN?

# 1: clone this repository.
# 2: edit .env file.
# 3: run npm install & npm run dev.
# 4: run migration command php artisan migrate.
# 5: run php artisan db:seed command
